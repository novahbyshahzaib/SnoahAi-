2. Service Description

SnoahAI is an AI chat assistant that connects to third-party AI platforms (Google Gemini, OpenRouter, Groq, Together AI, Mistral AI) using API keys that you provide. SnoahAI does not provide its own AI models.

3. API Keys & Data Storage

Your API keys and preferences are stored locally on your device using browser local storage. SnoahAI does not transmit your keys to any server other than the selected AI platform's official API endpoint.

  • Cookies Accepted: You may store multiple API keys per platform, enabling auto-rotation on rate limits for uninterrupted usage.
  • Cookies Rejected: Storage is limited to 1 API key per platform. Additional keys cannot be saved.

You are responsible for safeguarding your API keys and for any charges incurred through their use. Do not share your API keys publicly. You must comply with the terms of service of each third-party AI platform.

SnoahAI operates entirely in your browser. Chat history, settings, and API keys are stored in your browser's local storage and are not sent to SnoahAI's servers. Conversations are sent directly to the AI platform you have selected.

Who We Are

SnoahAI is a privacy-first AI chat assistant built by Shahzaib Muzaffar. Our mission is to provide a seamless, intelligent conversational experience that connects you to the world's leading AI platforms — all from a single, elegant interface that runs entirely in your browser.

What We Do

SnoahAI serves as a unified gateway to multiple AI platforms including Google Gemini, OpenRouter, Groq, Together AI, and Mistral AI. We provide features like multi-model support, voice input & output, image analysis, code highlighting, math rendering, and much more — all without requiring a backend server.

Our Philosophy

We believe your data belongs to you. SnoahAI stores everything locally in your browser — your API keys, chat history, settings, and preferences never leave your device. No accounts, no tracking, no servers collecting your conversations.

  • Universal Vision Support 🖼️ — Image upload now works on ALL 24 platforms that use the OpenAI-compatible API (NVIDIA, Groq, Together AI, Mistral, Claude, Hugging Face, DeepInfra, Ollama, Cerebras, SambaNova, Fireworks, xAI, Perplexity, NanoGPT, Unify, Kluster.ai, OpenRouter, and more). Vision availability depends on the selected model — if the model doesn't support images the platform returns a clear error.

Keys are now validated and sanitized on every load. Cookie-reject trimming now correctly covers all 24 platforms instead of just 5.
