Terms & Conditions

Last updated: March 2026

1. Acceptance of Terms

By accessing and using SnoahAI, you agree to be bound by these Terms and Conditions. If you do not agree, please do not use this application.

2. Service Description

SnoahAI is an AI chat assistant that connects to third-party AI platforms (Google Gemini, OpenRouter, Groq, Together AI, Mistral AI) using API keys that you provide. SnoahAI does not provide its own AI models.

3. API Keys & Data Storage

Your API keys and preferences are stored locally on your device using browser local storage. SnoahAI does not transmit your keys to any server other than the selected AI platform's official API endpoint.

  • Cookies Accepted: You may store multiple API keys per platform, enabling auto-rotation on rate limits for uninterrupted usage.
  • Cookies Rejected: Storage is limited to 1 API key per platform. Additional keys cannot be saved.

4. User Responsibilities

You are responsible for safeguarding your API keys and for any charges incurred through their use. Do not share your API keys publicly. You must comply with the terms of service of each third-party AI platform.

5. Privacy

SnoahAI operates entirely in your browser. Chat history, settings, and API keys are stored in your browser's local storage and are not sent to SnoahAI's servers. Conversations are sent directly to the AI platform you have selected.

6. Disclaimer

SnoahAI is provided "as is" without warranties of any kind. AI responses may contain inaccuracies. Do not rely on SnoahAI for medical, legal, or financial advice without consulting a qualified professional.

About Us

SnoahAI

Who We Are

SnoahAI is a privacy-first AI chat assistant built by Shahzaib Muzaffar. Our mission is to provide a seamless, intelligent conversational experience that connects you to the world's leading AI platforms — all from a single, elegant interface that runs entirely in your browser.

What We Do

SnoahAI serves as a unified gateway to multiple AI platforms including Google Gemini, OpenRouter, Groq, Together AI, and Mistral AI. We provide features like multi-model support, voice input & output, image analysis, code highlighting, math rendering, and much more — all without requiring a backend server.

Our Philosophy

We believe your data belongs to you. SnoahAI stores everything locally in your browser — your API keys, chat history, settings, and preferences never leave your device. No accounts, no tracking, no servers collecting your conversations.

Contact

SnoahAI is developed and maintained by Shahzaib Muzaffar. For feedback, suggestions, or issues, please reach out through our GitHub repository.

v2.0 Beta

🆕 What's New

  • User Name Personalization — New users are greeted with a name prompt on first visit. Your name is displayed in the sidebar and saved locally.
  • Improved TTS Voice — Text-to-Speech now uses a higher-quality voice (e.g., Google UK English Female) when available on your device.
  • TTS Auto-Stop Fix — TTS now properly stops at the end of the response instead of restarting. No more repeated playback.
  • Smooth Streaming Animation — Redesigned streaming cursor with smooth pulse animation and a progress bar above the input area for better visual feedback.
  • Floating Loading Indicator — The "SnoahAI is typing..." indicator now appears above the input bar, always visible during AI response generation.
  • Haptic Feedback — Subtle vibration feedback on mobile devices when AI starts responding and when streaming completes.
  • Stream Timeout Protection — Automatic timeout after 45 seconds of no data, preventing infinite loading states. Clear error message displayed.
  • Updated AI Knowledge — SnoahAI now knows about all 32 features of the app and can answer any question about how to use them.

📋 All Features

  • 5 AI Platforms — Google Gemini, OpenRouter, Groq, Together AI, Mistral AI. Switch in Settings.
  • Thinking Mode — 🧠 AI reasons step-by-step with a live collapsible thinking block.
  • Web Search — 🔍 Real-time Google Search grounding via Gemini. Works from any platform when enabled.
  • File Attachments — 📎 Upload images, PDFs, text files, CSVs, JSON, Markdown, and more. Multi-file support with preview strip.
  • Temp Chat — 🕐 Temporary conversations that won't be saved to history.
  • Response Length — Cycle between Normal, Short, and Detailed response lengths.
  • Templates — 🔖 Pre-saved prompt starters with custom template support.
  • Dark Mode — Full dark theme via ☰ menu toggle.
  • Chat History — Auto-saved locally with auto-titling.
  • Chat Tags — Organize chats by category (Biology, Medicine, Physics, Code, Math, Other).
  • Pinned Messages — 📌 Pin important AI responses for quick access.
  • Summarize Chat — AI-generated 3-5 bullet summary of conversations.
  • Search Chats — Search across all saved conversations.
  • Custom Instructions & AI Persona — Personalize AI behavior in Settings.
  • Themes — 4 color themes: Default, Ocean, Forest, Rose.
  • Slash Commands — Type / for quick shortcuts: /explain, /summarize, /translate, and more.
  • Code Runner — Run JavaScript and HTML code blocks inline.
  • TTS & Voice Input — 🔊 Read responses aloud with high-quality voice and 🎤 speak messages.
  • Multi-Key Pool — Add multiple API keys per platform with auto-rotation on rate limits.
  • Token Usage Tracking — Monitor your token usage per platform in Settings.
  • PWA Install — Install as a native-like app on Android home screen.
  • Contact Us — Reach the developer via Telegram or Instagram from the sidebar.
